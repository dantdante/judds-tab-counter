# Judds Tab Counter 💸

A simple Python app to track who owes you money.

## Features

* Add people and how much they owe
* Pay back debts
* Delete entries
* Automatically saves data
* Shows total money owed
* Clean dark-mode UI
* Terminal launcher

## How to Run

1. Make sure Python 3 is installed

2. Open terminal

3. Navigate to the folder:
   cd Desktop/Tab Counter

4. To Run:
   python3 tabcounter.py

## How It Works

* Data is stored in `tabs.json`
* Old data formats are automatically fixed
* The app sorts people by highest debt

## Controls

* Add: enter name + amount → click "add"
* Pay: select a person → enter amount → click "pay"
* Delete: select a person → click "delete"

## Notes

* If something breaks, delete `tabs.json` and restart
* First launch may require macOS permission (right click → open)
